### Added

## [0.1.0] - 2026-03-03

### Added
- CLI utility to extract interleaved multichannel WAV recordings into one mono WAV file per channel.
- Support for directory input with split WAV segments (e.g. 4GB WING X-LIVE files).
- GitHub Actions release workflow for Linux, macOS (Intel + Apple Silicon), and Windows.
